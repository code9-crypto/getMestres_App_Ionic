(ref.: https://angular.io/guide/setup-local)
Instalar Angular CLI
npm install -g @angular/cli

(ref.: https://angular.io/guide/setup-local)
Criar novo App
ng new my-app

(ref.: https://angular.io/guide/setup-local)

ng serve --open
